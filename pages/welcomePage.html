Welcome to NoHotMILFs

This app is just a GUI for yt-dlp that I made so I can easily download stuff for my PC's soundboard (Also because I wanted to have a go at using Electron)

You can download a given video in any of its available formats and resolutions, or just its audio only

Downloads will be placed in the users Downloads directory by default; this can be changed in the app's settings page

See the help page for more information (? icon in top right of main app window)
